# 🎮 Connect Four Game

A responsive Connect Four Game developed using HTML, CSS, and JavaScript.

# Features

- Two Player Gameplay
- Interactive 7×6 Board
- Horizontal, Vertical & Diagonal Win Detection
- Draw Detection
- Restart Game
- Responsive Design
- Move Counter
- Modern User Interface

# Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
